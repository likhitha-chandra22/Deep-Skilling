# Module 3 - Advanced SQL Using SQL Server

## Overview

This module covers advanced SQL concepts in Microsoft SQL Server, including database programming, performance optimization, stored procedures, user-defined functions, triggers, cursors, and exception handling.

### Exercise 1 - Ranking and Window Functions
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- Common Table Expressions (CTE)
- Top N Records per Category

### Exercise 2 - Indexes
- Non-Clustered Index
- Composite Index
- Covering Index
- Query Optimization

### Exercise 3 - Views
- Simple Views
- Join Views
- Computed Columns
- Employee Report View

### Exercise 4 - Stored Procedures
- Creating Procedures
- Altering Procedures
- Dropping Procedures
- Input Parameters
- Output Parameters
- Transactions

### Exercise 5 - User Defined Functions
- Scalar Functions
- Table-Valued Functions
- Nested Functions

### Exercise 6 - Triggers

### Exercise 7 - Cursors
- Simple Cursor
- Static Cursor

- Keyset Cursor

### Exercise 8 - Exception Handling
- TRY...CATCH
- THROW
- RAISERROR
- Nested TRY...CATCH

## Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- T-SQL

## Learning Outcomes

- Implemented advanced SQL queries using window functions and CTEs.
- Created and managed indexes to improve query performance.
- Developed views, stored procedures, user-defined functions, triggers, and cursors.
- Applied transactions and exception handling using TRY...CATCH, THROW, and RAISERROR.
- Strengthened SQL programming and database management skills.
