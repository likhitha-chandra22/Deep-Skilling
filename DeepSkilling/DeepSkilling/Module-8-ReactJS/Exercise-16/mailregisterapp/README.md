# Exercise 16 - React Form Validation

## Objective
Developed a React application to perform user registration with form validation using React Forms and Controlled Components.

## Features
- Registration form with:
  - Full Name
  - Email
  - Password
- Validations:
  - Full Name must contain at least 5 characters.
  - Email must contain '@' and '.'.
  - Password must contain at least 8 characters.
- Displays appropriate alert messages for invalid inputs.
- Displays "Valid Form" when all validations are successful.

## Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

## Project Structure

```
src/
│── App.js
│── Register.js
│── index.js
│── index.css
```

## How to Run

1. Open the project in VS Code.
2. Install dependencies:
   ```
   npm install
   ```
3. Run the application:
   ```
   npm start
   ```

## Learning Outcomes
- Understood React Forms and Controlled Components.
- Implemented form validation using event handlers.
- Performed validation using `onChange` and `onSubmit`.
- Validated Full Name, Email, and Password fields.
- Displayed validation messages using alert boxes.
