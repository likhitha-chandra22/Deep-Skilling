# Module 8 - ReactJS

## Exercise 4

### Objective

Implement React component lifecycle methods using a class component.

### Concepts Covered

- Class Components
- componentDidMount()
- componentDidCatch()
- Fetch API
- React State
- Render Method

### Files Created

- Post.js
- Posts.js

### Steps Performed

1. Created a React application named `blogapp`.
2. Created a `Post` class.
3. Created a `Posts` class component.
4. Initialized state with an empty posts list.
5. Implemented `loadPosts()` using the Fetch API.
6. Invoked `loadPosts()` inside `componentDidMount()`.
7. Displayed posts using the `render()` method.
8. Implemented `componentDidCatch()` for error handling.
9. Added the `Posts` component to `App.js`.

### Result

Successfully implemented React lifecycle methods to fetch and display blog posts from an API.
