# Module 8 - ReactJS

## Exercise 8

### Objective

Create a React application using State to count the number of people entering and exiting a mall.

### Concepts Covered

- React State
- Constructor
- setState()
- Event Handling
- Class Components

### Components Created

- CountPeople

### Files Created

- CountPeople.js
- App.js

### Steps Performed

1. Created a React application named `counterapp`.
2. Created a `CountPeople` class component.
3. Initialized `entrycount` and `exitcount` using state.
4. Implemented `updateEntry()` to increment entry count.
5. Implemented `updateExit()` to increment exit count.
6. Displayed the counts using two buttons.
7. Executed the application using `npm start`.

### Output

- Login button increments the entry count.
- Exit button increments the exit count.

### Result

Successfully implemented React State and event handling to update and display dynamic values.
