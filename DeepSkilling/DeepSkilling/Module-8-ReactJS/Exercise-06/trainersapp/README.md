# Module 8 - ReactJS

## Exercise 6

### Objective

Create a React application to manage trainer information using React Router for navigation.

### Concepts Covered

- React Router DOM
- BrowserRouter
- Routes and Route
- Link
- useParams
- Navigation
- Dynamic Routing
- React Components
- Mock Data

### Components Created

- Home
- TrainersList
- TrainerDetails

### Files Created

- trainer.js
- TrainersMock.js
- Home.js
- TrainersList.js
- TrainerDetails.js
- App.js

### Steps Performed

1. Created a React application named `trainersapp`.
2. Installed `react-router-dom`.
3. Created a `Trainer` model class.
4. Added mock trainer data in `TrainersMock.js`.
5. Created the Home component.
6. Created the TrainersList component to display trainer names.
7. Configured routing using BrowserRouter, Routes, and Route.
8. Implemented TrainerDetails using the `useParams` hook.
9. Displayed trainer details based on the selected trainer.
10. Executed the application using `npm start`.

### Output

- Home page
- Trainers List
- Trainer Details page with trainer information

### Result

Successfully implemented routing and navigation in a React application using React Router DOM and displayed trainer details using dynamic URL parameters.
