# Exercise 15 - Ticket Raising App

## Objective
Developed a React application to register employee complaints using React Forms and Controlled Components.

## Features
- Employee Name input field.
- Complaint textarea.
- Submit button with form handling.
- Generates a random Reference ID on submission.
- Displays a confirmation alert with employee name and reference ID.
- Implemented using React Hooks (`useState`).

## Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

## Project Structure

```
src/
│── App.js
│── ComplaintRegister.js
│── index.js
│── index.css
```

## How to Run

1. Open the project in VS Code.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the application:
   ```
   npm start
   ```

## Learning Outcomes
- Understood React Forms.
- Implemented Controlled Components using `useState`.
- Handled form submission using `handleSubmit`.
- Used textbox, textarea, and button input controls.
- Generated a complaint reference number after form submission.
