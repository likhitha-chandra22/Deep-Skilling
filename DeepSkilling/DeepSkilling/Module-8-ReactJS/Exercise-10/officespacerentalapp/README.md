# Module 8 - ReactJS

## Exercise 10

### Objective

Create a React application using JSX to display office space rental information.

### Concepts Covered

- JSX
- React Elements
- JavaScript Expressions
- Inline CSS
- Object Rendering
- Array Mapping

### Files Created

- App.js

### Steps Performed

1. Created a React application named `officespacerentalapp`.
2. Created a list of office space objects.
3. Displayed office details using JSX.
4. Rendered office images.
5. Used inline CSS to display rent in red when it is less than or equal to 60000 and green when it is greater than 60000.
6. Executed the application using `npm start`.

### Result

Successfully displayed office space details using JSX, JavaScript expressions, and inline CSS.
