# Module 8 - ReactJS

## Exercise 1

### Objective

Create the first React application using Create React App and display the message **"Welcome to the First Session of React"**.

### Concepts Covered

- React Installation
- Node.js and npm
- Create React App
- React Components
- JSX
- App Component
- npm start
- Hot Reloading

### Steps Performed

1. Installed Node.js and npm.
2. Created a React application using `npx create-react-app myfirstreact`.
3. Opened the project in Visual Studio Code.
4. Modified the `App.js` file.
5. Executed the application using `npm start`.
6. Verified the output in the browser at `http://localhost:3000`.

### Output

```
Welcome to the First Session of React
```

### Result

Successfully created and executed the first React application using Create React App.
