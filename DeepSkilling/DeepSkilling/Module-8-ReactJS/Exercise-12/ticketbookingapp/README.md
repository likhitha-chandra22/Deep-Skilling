# Module 8 - ReactJS

## Exercise 12

### Objective

Implement conditional rendering in a React application for a ticket booking system.

### Concepts Covered

- Conditional Rendering
- Functional Components
- useState Hook
- Event Handling
- Conditional Display
- Login and Logout Functionality

### Components Used

- LoginButton
- LogoutButton
- GuestGreeting
- UserGreeting
- Greeting

### Files Created

- App.js

### Steps Performed

1. Created a React application named `ticketbookingapp`.
2. Implemented Login and Logout buttons.
3. Used the `useState` hook to maintain login status.
4. Displayed the Guest page with flight details when the user is logged out.
5. Displayed the User page after successful login.
6. Switched between Guest and User pages using conditional rendering.
7. Executed the application using `npm start`.

### Output

- Guest users can view flight details and the Login button.
- Logged-in users can view the Welcome page and the Logout button.
- Clicking Login and Logout updates the page dynamically.

### Result

Successfully implemented conditional rendering in React using functional components, event handling, and the `useState` hook to switch between Guest and User views.
