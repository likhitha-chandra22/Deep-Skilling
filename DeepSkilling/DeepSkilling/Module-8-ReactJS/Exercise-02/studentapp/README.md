# Module 8 - ReactJS

## Exercise 2

### Objective

Create a React application named **studentapp** with multiple class components and render them from the main App component.

### Concepts Covered

- React Components
- Class Components
- render() Method
- Component Composition
- React Imports and Exports

### Components Created

- Home
- About
- Contact

### Steps Performed

1. Created a React application using Create React App.
2. Created a `Components` folder under `src`.
3. Added `Home.js`, `About.js`, and `Contact.js`.
4. Implemented each as a class component.
5. Imported and rendered all components in `App.js`.
6. Executed the application using `npm start`.

### Output

- Welcome to the Home page of Student Management Portal
- Welcome to the About page of the Student Management Portal
- Welcome to the Contact page of the Student Management Portal

### Result

Successfully created multiple React class components and rendered them in the React application.
