# Exercise 17 - Fetch User App

## Objective
Developed a React application that retrieves user information from a REST API and displays the user's title, first name, and profile image.

## Features
- Fetches user details from Random User API.
- Uses `componentDidMount()` for API calls.
- Displays user's title and first name.
- Displays user's profile image.
- Shows loading message while fetching data.

## Technologies Used
- React JS
- JavaScript (ES6)
- Fetch API
- REST API

## Project Structure

```
src/
│── App.js
│── Getuser.js
│── index.js
│── index.css
```

## How to Run

1. Install dependencies:
   ```
   npm install
   ```
2. Start the application:
   ```
   npm start
   ```

## Learning Outcomes
- Understood REST API integration in React.
- Used Fetch API to retrieve data.
- Implemented `componentDidMount()`.
- Managed component state and asynchronous data loading.
