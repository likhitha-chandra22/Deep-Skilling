# Module 4 – NUnit and Moq

## Overview

This module focuses on automated unit testing using the NUnit framework and mocking external dependencies using the Moq framework. It covers writing effective unit tests, validating application logic, and isolating external components using mock objects.

## Learning Objectives

- Understand Automated Testing
- Understand Unit Testing
- Write NUnit Test Cases
- Use Assertions
- Test Exceptions
- Test Collections
- Use TestCase and TestCaseSource
- Mock External Dependencies
- Apply Dependency Injection
- Understand Constructor Injection
- Learn Test Driven Development (TDD)
